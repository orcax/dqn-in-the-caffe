DQN-in-the-Caffe
================

DQN-in-the-Caffe is an implementation of Deep Q-Network using Caffe.

How to run:
- compile caffe
cd caffe
make all
- compile xitari
cd xitari
make
- compile dqn
cmake .
make 

Algorithm Details
--

See http://www.cs.toronto.edu/~vmnih/docs/dqn.pdf for the details of DQN.
